🔥 Flame Language

Created by Daniel Ricardo Maranhão Santana

Overview

Flame is a lightweight scripting language designed for data analysis, automation, and fire risk modeling. Its syntax is clean and intuitive — ideal for rapid prototyping and operational scripting.

🔥 Interpreter

The official Flame interpreter is built in Python. You can run it directly:

python flame_interpreter.py

Or use it inside your code:

from flame_interpreter import run_flame_code
run_flame_code(open("examples/hello.flame").read())

📂 CSV Example

Flame can process fire risk data line-by-line from a CSV:

# csv-fire-risk.flame
let linhas = [
  "Concelho,Temp,Humidade,FWI",
  "Proença-a-Nova,38,18,72",
  "Castelo Branco,34,22,60",
  "Sertã,40,15,80"
]

let header = linhas[0].split(",")

for i in 1..len(linhas)-1:
  let partes = linhas[i].split(",")
  let concelho = partes[0]
  let temp = to_number(partes[1])
  let humidade = to_number(partes[2])
  let fwi = to_number(partes[3])

  if temp > 37 and humidade < 20 and fwi > 70:
    print("🚨", concelho, "→ EXTREME RISK")
  elif fwi > 60:
    print("⚠️", concelho, "→ HIGH RISK")
  else:
    print("✅", concelho, "→ LOW RISK")
